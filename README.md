# SpringCloudDemo


虽然项目里有用到SpringCloud但是一直没有系统的去整理学习过，最近在系统的走遍各种知识同时会写点Demo

### 简介

在微服务架构中，需要几个基础的服务治理组件，包括**服务注册与发现、服务消费、负载均衡、断路器、智能路由、配置管理**等，由这几个基础组件相互协作，共同组建了一个简单的微服务系统。一个简答的微服务系统如下图：

### Spring Cloud的核心特性： 

- 分布式/版本化配置
- 服务注册和发现
- 路由
- 服务和服务之间的调用
- 负载均衡
- 断路器
- 分布式消息传递
